Student Exam Performance Predictor

This web application uses a machine learning model to estimate a student's math score based on their academic background and test preparation details.

Model Details

  • Problem Type: Regression
  • Target Variable: Math Score
  • Dataset: Student Exam Performance Dataset
  • Best Model Used: CatBoost Regressor
  • Model Performance (R² Score): 0.8516
  • Technologies: Python, Scikit-learn, CatBoost, Flask, HTML/CSS
